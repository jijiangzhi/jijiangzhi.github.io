一周iOS资源精选【第1期】

2016-02-27 | 分类于 周刊

技匠 为你精心收集整理一周内国外资深设计师与开发人员推荐的优秀IOS设计/开发资源，涵盖工具、设计、教程、文章、书籍等各个方面。使你在IOS设计/开发领域始终与世界保持同步。

本周带来第一期精彩内容：
（由于是第一期，花了更多时间准备，内容稍有滞后，请大家见谅 ^__^）

30 Days of Swift

一个设计师用一个月时间学习并使用Swift完成了30个IOS小应用，并将它们分享在GitHub上。这些小应用都各不相同，涵盖了Swift开发的各个方面，非常值得学习。

Sketch 3.6 Beta

Sketch 3.2 beta版本已经开放下载试用，修复了已经存在很久的line-height问题，并且整个工具的性能得到了很大的提高。了解更多新特性可以参考推酷上的 《Sketch 3.2 beta新特性》这篇文章。

iOS Interfaces with Xcode Playgrounds

这是一篇关于Xcode新特性——Interactive Playgrounds 的快速指南。上面的示例非常详细，通过这篇文章你可以学会如何使用Interactive Playgrounds来定制自己的UI组件。

Free Chat UI kit for Photoshop and Sketch

来自InVision的一套非常漂亮的UI Kit，它是完全免费的，并且同时提供了Sketch和PS两个版本。

Can you Code this UI Concept?

来自Medium的一篇很有意思的文章，讲述如何将一个漂亮的UI设计转换为原型的，并分享了相关代码。使用到Jade，SCSS等技术。

Mobile Prototyping with Xcode

越来越多的设计师开始直接使用XCode进行原型设计和开发。这篇文章一步一步告诉你他们是怎么做得。

Circle Menu

一个用Swift开发的非常有趣的环形菜单动效，在GitHub上可以得到它的源码。

ProtoSketch – Graphic Design and Prototypes Made Easy

ProtoSketch是iPad上最接近Sketch的设计工具，能够编辑矢量图，导出SVG，甚至还有IOS 9 UI模板可供使用。

Best Articles of Medium 2015

如果你想要一些在线下阅读的设计文章，那么通过这个链接可以下载2015年Medium上的最佳文集。

Sketch Tutorial for iOS Developers

一个非常棒的面向开发者的Sketch教程，展示了如果使用Sketch进行设计并设计转换到Xcode中的。

Social Media Whitewash

这篇文章告诉我们设计不总是前后一致的，我们要做的是不断提高它的辨识度。

Zeplin Now Generates CSS

Zeplin仍在不断地改进，已经可以通过它自动生成CSS代码。

Great Books for Designers to Read in 2016

一份Robert Greene推荐的2016年设计师阅读书单，非常棒，每个设计师都应该去读它们。

Frontend Design

一篇关于前端设计的非常有意思的文章，告诉我们为什么将设计师和开发人员结合在一起是有意义的。
